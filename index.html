Connect to Infor Data Lake

Most Infor Data Lake users will already have an Infor Compass JDBC Driver app. Infor Compass is used to send, extract, and query data to and from the Infor Data Lake.

  1. Login to your Infor ION API application through ION Desk.

  2. Select Authorized Apps.

  3. Open the app named “Infor Compass JDBC Driver”.

  4. Click on the Download Credentials button to open a dialog box.

  5. In the User Name field, type the first few characters of your Infor Ming.le user name. Then select your user name from the type-ahead list.

    Note: You must select the user name from the type-ahead list because the type-ahead values are the user names registered in Infor Ming.le.

  6. Click Download. Two files will be downloaded. Save the Infor Compass JDBC Driver.ionapi file and the infor-compass-jdbc-version.jar file to a safe and secure location. You will upload the Infor Compass JDBC Driver.ionapi in the form below.

If you do not have the Infor Compass JDBC Driver app already installed then follow these steps.

You will need to have the role of IONAPI-Administrator.

  1. Login to your Infor ION API application through ION Desk.

  2. Select Authorized Apps.

  3. Click on the plus icon to add a new app (non-Infor authorized application).

  4. Name the app “Infor Compass JDBC Driver”. The name must be exact.

  5. Use “Infor Compass JDBC Driver” for the description as well.

  6. Select the Backend Service type

  7. Save the new app

  8. From the Authorized Apps menu, open the new Infor Compass JDBC driver application.

  9. Click on the Download Credentials button to open a dialog box.

  10. Select Create Service Account.

  11. In the User Name field, type the first few characters of your Infor Ming.le user name. Then select your user name from the type-ahead list.

    Note: You must select the user name from the type-ahead list because the type-ahead values are the user names registered in Infor Ming.le.

  12. Click Download. Two files will be downloaded. Save the Infor Compass JDBC Driver.ionapi file and the infor-compass-jdbc-version.jar file to a safe and secure location. You will upload the Infor Compass JDBC Driver.ionapi in the field below.
